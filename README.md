🎲 Sorteador de Números - Um projeto dinâmico e seguro para sortear números aleatórios!

Neste projeto, desenvolvi uma aplicação que sorteia números aleatórios dentro de um intervalo definido pelo usuário. 🚀 O grande destaque foi implementar proteções inteligentes contra erros comuns: primeiro, garanti que nunca sejam sorteados números repetidos utilizando um loop com validação; depois, adicionei uma verificação crucial para evitar loops infinitos, validando se a quantidade de números solicitados não ultrapassa o intervalo disponível. ✅ Essas validações garantem que o programa funcione de forma segura e eficiente, prevenindo situações onde o usuário poderia solicitar mais números únicos do que é possível sortear no intervalo escolhido. 💪

🛠️ Tecnologias e Ferramentas Utilizadas:

- HTML5 - Estrutura e markup da aplicação
- CSS3 - Estilização e design responsivo
- JavaScript - Lógica de programação e funcionalidades
- Live Server - Servidor local para desenvolvimento
- Git & GitHub - Controle de versão e repositório
